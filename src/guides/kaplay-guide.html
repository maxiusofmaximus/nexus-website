Kaplay Engine

Motor 2D JavaScript moderno y fácil de usar para crear juegos web

JavaScript Fácil 2D

Configuración Inicial

Instalación Básica
// Incluir Kaplay en tu proyecto
import kaplay from "kaplay";

// Inicializar el motor
const k = kaplay({
  width: 800,
  height: 600,
  background: [0, 0, 0]
});
Configuración Avanzada
const k = kaplay({
  width: 1024,
  height: 768,
  debug: true,
  background: [135, 206, 235],
  global: true
});

Comandos Principales

Crear Objetos
add([
  sprite("player"),
  pos(100, 100),
  area(),
  "player"
]);
Cargar Sprites
loadSprite("player", "sprites/player.png");

loadSprite("enemy", {
  src: "sprites/enemy.png",
  sliceX: 4
});
Movimiento
player.move(100, 0);

onKeyDown("left", () => {
  player.move(-SPEED, 0);
});

Eventos y Controles

Controles de Teclado
onKeyPress("space", () => {
  // Saltar
  player.jump();
});

onKeyDown("right", () => {
  player.move(SPEED, 0);
});
Colisiones
player.onCollide("enemy", (enemy) => {
  // Lógica de colisión
  destroy(enemy);
  addKaboom(enemy.pos);
});
